Subscription Feed Filter for YouTube
GitHub 

This extension brings filter options similar to those on the YouTube home screen to pages like Subscriptions and Library.

<Filters>

The following filters are available:
- Live
- Streamed
- Videos
- Shorts
- Scheduled
- Notifications On
- Notifications Off
- Unwatched
- Watched

By enabling multi-selection in the settings, you can combine multiple filters. For example, selecting both "Live" and "Streamed" will display videos that are either live or have already been streamed.

You can also customize the order and display names of the filters in the settings.


<Keyword Search>

You can filter videos using keywords. For example, entering "News" will display videos with "News" in the title.

Keyword search also supports filtering in the Sidebar Subscriptions, Notifications, and Save to playlist sections.

Supported Platform: YouTube.com

YouTube Live Stream Latency Mitigator
GitHub 

This extension temporarily increases the playback speed of YouTube live streams with DVR enabled when a delay is detected (i.e., the red "LIVE" indicator turns gray), helping you catch up to real time.

When "Aggressive mode" is enabled in the settings, the extension continues increasing playback speed to catch up to real time as long as there is buffer, even if YouTube has not detected a delay. Aggressive mode also works on YouTube live streams where DVR is disabled.

Additional features include:
- Displaying the current playback speed
- Displaying the number of seconds delayed
- Displaying the remaining buffer size
- Predicting and showing the time you will catch up to real time at the current speed
- Displaying the elapsed time of the live stream
- Copying the current timestamp to the clipboard by clicking the elapsed time display

Supported Platforms: YouTube.com, YouTube Embedded Player (e.g. Holodex)

Auto Pan by Window Position
GitHub 

This extension controls audio panning, enabling sound to be heard from the left when the window is positioned to the left of the display's center and from the right when it's on the right side.
It may not function on sites with CORS configured.

Supported Platforms: YouTube.com, YouTube Embedded Player (e.g. Holodex), X (formerly Twitter)
CAUTION: On Twitch, the audio may become muted.

Improve YouTube Live Chat
GitHub 

This extension provides the following features to improve YouTube live chat:
- Displays each message in a single line. This ensures that even long messages spanning multiple lines take up uniform space. Hovering over a message will display its full content as usual.
- Hides icons, names, and badges from messages.
- Differentiates message text and background colors based on user type: general users, members, moderators, and owners.

Supported Platforms: YouTube.com, Holodex
